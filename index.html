Entendendo HTML

HTML é a sigla para Hypertext Markup Language, ou seja, Linguagem de Marcação de Hipertexto. É uma linguagem de marcação utilizada para criar páginas web.

Veja abaixo um resumo de algumas das principais tags HTML:

  • Títulos
  • Parágrafos
  • Texto em destaque
  • Listas
  • Links
  • Linhas horizontais
  • Texto subscript
  • Texto superscript
  • Citações
  • Fonte
  • Texto riscado
  • Abreviações

Títulos

As tags HTML de títulos vão desde <h1> até <h6>, sendo que <h1> é o título principal da página e <h6> é o subtítulo de menor nível.

Parágrafos

A tag <p> é usada para criar parágrafos no HTML.

Texto em destaque

As tags <mark>, <small>, <i>, <u>, <strong>são usadas para destacar partes do texto:

  • <mark> é usada para destacar um trecho do texto
  • <small> é usada para especificar texto menor que o texto normal
  • <i> é usada para especificar texto em itálico
  • <u> é usada para sublinhar o texto
  • <strong> é usada para especificar texto em negrito

Listas

As tags <ol> e <ul> são usadas para criar listas ordenadas e não ordenadas, respectivamente. Já a tag <li> é usada para criar um item na lista.

Links

A tag <a> é usada para criar links para outras páginas ou para acessar recursos externos, como imagens e arquivos de áudio. O atributo "href" é usado para especificar o endereço para onde o link deve apontar.

Linhas horizontais

A tag <hr> é usada para criar uma linha horizontal no HTML, separando partes do texto.

Texto subscript

A tag <sub> é usada para criar texto subscript, ou seja, texto alinhado abaixo da linha de base do texto normal.

Texto superscript

A tag <sup> é usada para criar texto superscript, ou seja, texto alinhado acima da linha de base do texto normal.

Citações

A tag <blockquote> é usada para criar citações longas, geralmente de outros autores.

Formatação de fonte

A tag <font> é usada para especificar a formatação da fonte, como tamanho, cor e tipo de fonte. Embora ainda seja suportada, essa tag é considerada obsoleta e não é recomendada para uso.

Texto deletado

A tag <del> é usada para especificar texto deletado, geralmente com uma linha horizontal riscada através do texto.

Abreviações

A tag <abbr> é usada para especificar abreviações e siglas. O atributo "title" pode ser usado para especificar a forma completa da abreviação.
